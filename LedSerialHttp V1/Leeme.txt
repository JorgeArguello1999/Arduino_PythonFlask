
==> En arduino usa el script: ledSerialV3.ino 
    Ya que este guarda la data del ultimo estado en memoria y te va a permitir siempre despues de un reinicio o nueva comunicación serial tener el estado seteado en la salida

=> Luego en python tienes el scripts: cuminicacionSerial.py
   Este te vaa servir para entender como hacer la conexion serial desde python al arduino.

==> Luego tienes el scripts: LedSerial.py
    Este es el que se va a encargar de tener 3 funciones para comunicarse con el arduino segun sea el caso

==>Por ultimo tienes los archivos:
==> script: receptorLastValue.py
    Este va abri un servidor web te permite entrar a una pagina y a partir de ahí configurar el estado del LED o saber su estado.

==> HTML en la carpeta templates: index.html
    Este es el codigo HTML que te estara mandando la aplicación cada vez que la uses para enviar un comando o saber el estado del LED.
